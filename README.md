SADOJRP is a FiveM content package centered around a San Andreas Department of Justice concept. It is aimed at roleplay servers that want a legal-roleplay presentation, with content arranged to reinforce a DOJ-style identity and a more deliberate in-world structure.

The project is focused on server-side roleplay usage instead of broad gameplay automation. Its purpose is to help define the look and organization of a DOJ-themed content hub, so a law-and-order environment can be presented more clearly inside a FiveM roleplay setup.
